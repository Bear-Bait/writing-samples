# Writing Portfolio: Technical and Creative

This repository contains a dual-track portfolio demonstrating high-level technical analysis and journalistic expertise.

## Part 1: Broadcast Infrastructure Incident Reports
A collection of real-world Incident Reports (Post-Mortems) and Root Cause Analyses (RCA) for a macOS-based broadcast automation system. These documents showcase systematic debugging of hardware/software failures in a 24/7 production environment.

### Technical Skills Demonstrated
- Log Analysis: Deep inspection of unified system logs (log show), kernel buffers (dmesg), and application error logs.
- Network Troubleshooting: Diagnosing socket exhaustion (ADDRINUSE), hardware bottlenecks, and DNS resolution hangs.
- Hardware Auditing: Verification of link speeds and bus topology using system_profiler and ioreg.
- Remediation: Implementing structural changes to prevent recurring failures.

### A Note on Document Design
With a professional background spanning a decade in high-end publishing—utilizing Adobe InDesign for Art Journals and Lacanian Ink—I have since transitioned to LaTeX for all technical documentation. I prioritize not only the diagnostic accuracy of my reports but also the typographic precision and structural clarity of the final output. This collection reflects a commitment to the aesthetic standards of professional publishing applied to the rigors of systems administration.
