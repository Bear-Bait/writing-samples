* Root Cause Analysis

** SUNDAY-WEEK-1 (Confirmed --- Fixes Verified Working SUNDAY-WEEK-2)

Three factors converged; all identified. The SUNDAY-WEEK-2 clean run confirms the
applied fixes were effective.

*1. DNS hang (immediate trigger)*

Radiologik's embedded VLC used a blocking =getaddrinfo()= call with no timeout.
When =mDNSResponder= reloaded at 08:09:30, the lookup hung indefinitely, corrupted
memory state, and crashed the process (=SIGSEGV= in =vlc_getaddrinfo=).

*2. CPU saturation from chronic missing-asset errors (enabling factor)*

A promo file absent for 2+ months caused 35+ scheduler retries at 40-minute
intervals. Combined with BlackHole→stream sample rate resampling (48kHz→44.1kHz,
~7% CPU overhead), baseline CPU was 12--15% before the incident.

*3. Empty AppleScript fallback (recovery gap)*

=DJEvent-PlayingSilenceFor8Seconds.applescript= contains only a comment --- no
executable code. Software recovery was inert.

** SUNDAY-WEEK-2 (Operational)

No technical failure on our systems. Stream gap was due to Show-Host-A starting
late.

** Legacy Path Poisoning --- Segment 446

Segment 446 hard-codes paths to =/Users/[LEGACY-USER-PATH]/= --- a user directory
that does not exist on the current Automation Node host.
